# MTABI — Novo logo (2026)

## Conceito
Ícone de blocos modulares em progressão — referência direta ao "skyline" do logo
anterior, mas reinterpretado como dados/código, alinhado ao posicionamento de
autoridade em desenvolvimento de soluções de tecnologia aplicadas a negócios.

## Paleta
- Fundo escuro principal: `#13151A`
- Âmbar (cor de destaque/identidade): `#E8A33D`
- Branco quebrado (texto e blocos claros): `#F2F0EA`

## Tipografia
**Space Grotesk Bold** — geométrica, técnica, usada por diversas empresas de
tecnologia/dev tools como referência visual de categoria.

A fonte está incluída na pasta `fonte/SpaceGrotesk-Bold.ttf`. Para o texto
aparecer corretamente em qualquer programa (Figma, Canva, Illustrator,
navegador), instale essa fonte no seu computador antes de abrir os arquivos:
- **Windows**: clique duplo no arquivo `.ttf` → Instalar
- **Mac**: clique duplo no arquivo `.ttf` → Instalar fonte
- Caso não instale, os arquivos abrem normalmente mas o texto "MTABI" pode
cair para uma fonte padrão do sistema (Arial) em vez da Space Grotesk.

Se for usar em site/web, baixe gratuitamente em fonts.google.com/specimen/Space+Grotesk
e carregue via Google Fonts (`@import` ou `<link>`).

## Arquivos incluídos

| Arquivo | Uso |
|---|---|
| `mtabi-logo-fundo-escuro.svg` | Versão principal, fundo escuro — uso padrão |
| `mtabi-logo-fundo-claro.svg` | Para colocar sobre branco/papel/documentos claros |
| `mtabi-logo-transparente.svg` | Para sobrepor em fotos, banners, qualquer fundo |
| `mtabi-logo-mono-branco.svg` | Tudo branco — para fundos coloridos ou imagens escuras |
| `mtabi-logo-mono-preto.svg` | Tudo preto — para impressão ou fundos muito claros |
| `mtabi-icone-apenas.svg` | Só o ícone, sem texto — para favicon, avatar de perfil, WhatsApp |

Todos os arquivos são vetoriais (SVG) — podem ser redimensionados para qualquer
tamanho (de favicon a banner gigante) sem perder qualidade, e podem ser abertos
e editados em Figma, Illustrator, Inkscape ou Canva.
